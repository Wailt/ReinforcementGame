  Цель проекта - создание игры, обьединяя методы математики и машинного обучения 
для посредственного влияния игрока на развитие персонажа и противников (агентов).

Особенности:
0) Игрок не может управлять сущностями непосредственно.
1) Нет предобученного ни главного героя (гг), ни противников. Всех агентов игрок обучает самостоятельно.
2) Цель игрока - обучить существо: строить и влиять на окружающею среду таким образом, что бы 
  получить от агентов развитие необходимых навыков.
3) Обученные агенты состязаются между собою для развития своих навыков 
    (или объединяются в команды для состязаниями с другими командами)
4) В последствии - публикация наиболее удачно тренированных сущностей/рейтинга игроков/соревнования команд разных игроков. 

5) Мозг сущности - продвинутый алгоритм: Нейронная сеть, многорукие бандиты и т.д... 
   Главное, что бы сущности обучались по ходу игры.
   Вполне возможно, что если игрок не удовлетворён алгоритмами обучения "по умолчанию", он будет писать свой по предоставленному интерфейсу
  
В перспективе, этот проект можно использовать как:
1) обучающую платформу для всех желающий, кто изучает "исскуственный интелект"
2) Бизнес идея игры (продажа)
3) Привлечения широкого круга людей для разработки новых алгоритмов для обучения с подкреплением.




В конце-концов проект должен иметь вид:
 1) Среда для обучения и соревнований, которую создаёт игрок.
    для обучения:
    - разнообразие "строительных блоков"
    - удобный интерфейс создания и редактироваани
    для соревнований:
    - Слечайная генерация
 
 2) Агенты, на которых влияет среда
    - Набор навыков и атрибутов
    - Принятие решений агентом и стратегическое мышление группы агентов на основе 
      приобретённого опыта и априори выданых аттрибутов.
  
 3) Внешний вид проекта:
    - Графический интерфейс.
    - Удобное представление навыков и атрибутов группы агентов
    - Логгирование боя
    - Возможность првести обучение агентов минуя графическое отображение.
    
 4) Мультиплеер:
    - импорт/експорт комманд для проведения соревнований
    - онлайн таблица (рейтинг)
 
 5) Интерфейс для написания модификаций:
    Вход - группы агентов
    - В центре - алгоритм, и функционалы качества которые реализует игрок. 
    - Алгоритм и функционалы влияют на оценку ситуации группой агентов.  
    Выход - принятое стратегическое решение 
    (На деле, у каждого решения, что принимает алгоритм, 
     будет набор функционалов качества, которые на него влияют.
     "Мозг" агента будет обучаться выбирать необходимые для него функционалы)
