  Цель проекта - создание игры, обьединяя методы математики и машинного обучения 
для посредственного влияния игрока на развитие персонажа и противников.

Особенности:
0) Игрок не может управлять сущностями непосредственно.
1) Нет предобученного ни главного героя (гг), ни противников. Все сущности игрок обучает самостоятельно.
2) Цель игрока - обучить существо: строить и влиять на окружающею среду таким образом, что бы 
  получить от существа развитие необходимых навыков.
3) Обученные сущности состязаются между собою для развития своих навыков 
    (или объединяются в команды для состязаниями с другими командами)
4) В последствии - публикация наиболее удачно тренированных сущностей/рейтинга игроков/соревнования команд разных игроков. 

5) Мозг сущности - продвинутый алгоритм: Нейронная сеть, многорукие бандиты и т.д... 
   Главное, что бы сущности обучались по ходу игры.
   Вполне возможно, что если игрок не удовлетворён алгоритмами обучения "по умолчанию", он будет писать свой по предоставленному интерфейсу
  
В перспективе, этот проект можно использовать как:
1) обучающую платформу для всех желающий, кто изучает "исскуственный интелект"
2) Бизнес идея игры (продажа)
3) Привлечения широкого круга людей для разработки новых алгоритмов для обучения с подкреплением.
